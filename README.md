Connect 4 AI Agent

A Connect 4 AI project developed at IIT Roorkee using supervised learning and deep reinforcement learning techniques. The project implements multiple intelligent agents and trains them through self-play simulations.

Features
Human vs AI gameplay
Random, ANN, and Reinforcement Learning agents
Self-play training
Deep Q-Network (DQN) implementation
Dueling Double DQN architecture
Experience replay and target networks
Performance evaluation framework
Technologies Used
Python
TensorFlow
Keras
NumPy
Results
Achieved near-100% win rate against random agents.
Dueling Double DQN achieved approximately 83% win rate.
Installation
